¿Cual es el objetivo?

El objetivo de esta práctica es desarrollar una página web que a través de Javascript incluya funcionalidades dinámicas.

Estructura de la Pagina

Elementos del menu

Pagina interactiva

La estructura de la página web deberá dividirse en 4 apartados: Cabecera, Menú, Lista y Estadísticas

El comportamiento de la página web se basará en modificar el DOM, añadiendo, modificando o eliminando elementos de una lista. Además, la página incluirá un apartado de estadísticas.

El funcionamiento del menú será el siguiente:
La primera opción será Mostrar/Ocultar Lista. Si la lista está visible, el texto será “Ocultar Lista”, en caso contrario será “Mostrar Lista”. Al pulsar Ocultar lista la lista se ocultará y se modificará el texto por “Mostrar Lista”, al pulsar Mostrar Lista, la lista se mostrará y se modificará el texto por “Ocultar Lista”.

Si la lista está visible las opciones de Añadir un elemento, Eliminar elemento, o Intercambiar elementos también deberán de estar visibles. Si la lista está oculta, no se verán esas opciones, y en el Menú, solo aparecerá la opción “Mostrar lista”. En el momento en el que se muestre la lista, se mostrarán también las opciones previamente mencionadas.

Funcionamiento de las opciones normales

  • Añadir un elemento: Al pulsar esta opción se añadirá un string aleatorio de entre 8 y 12 caracteres (también aleatorios) al final de la lista
  • Eliminar último: Al pulsar esta opción se eliminará el último elemento de la lista.
  • Intercambiar: Al pulsar esta opción se intercambiarán el primer y el último elemento de la lista

Opciones extras

  • “Añadir elemento por teclado”: Al pulsar esta opción se pedirá al usuario que introduzca el String que se quiere añadir a la lista, preferiblemente a través de un input text (aunque también se puede hacer a través de un prompt), que posteriormente se añadirá al inicio de la lista.
  • “Eliminar por posición”: Al pulsar esta opción se pedirá al usuario que introduzca la posición del elemento que se quiere eliminar, y se borrará dicho elemento de la lista
  • “Intercambiar por posición”: Al pulsar esta opción se pedirá al usuario que introduzca las dos posiciones de los dos elementos que se quieren intercambiar, y se intercambiarán en la lista
